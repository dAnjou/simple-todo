# simple-todo

## Install (Ubuntu 11.10 or higher)

Install `couchdb`, `python-couchdbkit` and `python-bottle`.

Clone repository.

## Run

Change into simple-todo's directory.

Run `python todo.py`.

## Use

#### Create a todo list

[http://localhost:8080/](http://localhost:8080/)

#### See all todo lists

[http://localhost:8080/l/](http://localhost:8080/l/)
